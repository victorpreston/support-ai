# Support AI - Customer Support Platform

A modern AI-powered customer support platform built with Next.js, TypeScript, and Tailwind CSS.

## Features

- AI-powered chatbot
- Analytics dashboard
- Conversation management
- Knowledge base
- AI configuration panel
- Performance monitoring

## Getting Started

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build

# Start production server
pnpm start
```

## Tech Stack

- **Framework:** Next.js 15
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Components:** Radix UI
- **Charts:** Recharts
- **Package Manager:** PNPM
